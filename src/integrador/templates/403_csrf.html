403 - Acesso Negado
A verificação de segurança CSRF (Cross-Site Request Forgery) falhou. A requisição foi bloqueada para proteger o sistema.
Motivo da falha:

{{ reason|default:"Token CSRF ausente ou inválido" }}

O que fazer?

  • Se você está integrando com esta API, certifique-se de incluir o token CSRF nas suas requisições
  • Verifique se o cookie de CSRF está sendo enviado corretamente
  • Confirme que a URL de origem está na lista de origens confiáveis (CSRF_TRUSTED_ORIGINS)
  • Para requisições POST/PUT/DELETE, inclua o header X-CSRFToken ou o campo csrfmiddlewaretoken
  • Se o problema persistir, entre em contato com o administrador do sistema
